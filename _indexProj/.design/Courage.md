# Courage (勇气) 系统设计文档

## 1. 系统概述

勇气（Courage）是殖民者的一项可成长属性，类似于射击、格斗、建造等技能。勇气等级越高，殖民者在战斗中受伤时无畏值的减少越少。

## 2. 核心组件

### 2.1 CourageComponent

**职责**: 管理殖民者的勇气等级和经验值

**关键属性**:
| 属性 | 类型 | 说明 |
|------|------|------|
| `levelInt` | int | 当前勇气等级（1-20） |
| `xpSinceLastLevel` | float | 当前等级已获得经验 |
| `MinLevel` | int | 最低等级（1） |
| `MaxLevel` | int | 最高等级（20） |

**核心方法**:
| 方法 | 功能 |
|------|------|
| `Initialize()` | 初始化勇气等级为基础值 |
| `AddXP()` | 添加经验值 |
| `CheckLevelUp()` | 检查是否升级 |
| `XpRequiredToLevelUpFrom()` | 计算升级所需经验 |

### 2.2 CourageComponent 的显示

勇气属性应显示在殖民者的技能面板中，与射击、格斗等技能并列。

## 3. 数值配置

所有配置集中在 `Define` 类中：

| 配置项 | 值 | 说明 |
|--------|-----|------|
| `CourageBaseLevel` | 1 | 初始等级 |
| `CourageMaxLevel` | 20 | 最高等级 |
| `CourageBaseXP` | 0f | 初始经验 |
| `CourageXPRequiredPerLevel` | 1000f | 基础升级经验 |

## 4. 核心逻辑

### 4.1 经验曲线

使用 SimpleCurve 定义升级所需经验：

| 等级 | 所需经验 |
|------|---------|
| 1→2 | 1,000 |
| 10→11 | 10,000 |
| 20→21 | 30,000 |

### 4.2 对无畏值的影响

勇气等级越高，受伤时无畏值减少越少：

```
无畏值减少系数 = 1 - (勇气等级 - 1) / 19 × 0.5
实际减少值 = 基础减少值 × 减少系数
```

勇气等级越高，恢复速度越快：

```
恢复速率 = 基础恢复速率 × (1 + (勇气等级 - 1) / 19 × 0.5)
```

### 4.3 经验获取

经验获取来源（待实现）：
- 成功击杀敌人
- 承受伤害并存活
- 完成战斗任务

## 5. 升级机制

当累计经验达到当前等级所需经验时：
1. 等级提升 1
2. 剩余经验 = 当前经验 - 升级所需经验
3. 下一等级所需经验增加

## 6. 数据持久化

通过 `IExposable` 接口实现存档：

```csharp
public void ExposeData()
{
    Scribe_Values.Look(ref levelInt, "levelInt", Define.CourageBaseLevel);
    Scribe_Values.Look(ref xpSinceLastLevel, "xpSinceLastLevel", Define.CourageBaseXP);
}
```

## 9. 未来扩展

### 9.1 勇气技能特性

| 等级 | 特性 |
|------|------|
| 5 | 受伤时无畏值减少 -10% |
| 10 | 受伤时无畏值减少 -25% |
| 15 | 受伤时无畏值减少 -40% |
| 20 | 受伤时无畏值减少 -50% |

### 9.2 勇气相关心情效果

- 勇气等级低的殖民者在战斗中更容易产生恐惧
- 勇气等级高的殖民者在战斗后可能获得"战斗经验"正面心情
